# Timgiasu-Backend

## What is it?

Timgiasu is a platform where students can help students teachers knows each other and made appointments for the class with the teachers online or offline.

## Backend tools?

- Nodejs
- MongoDB to store data
- bcrypt,helmet to improve the cybersecurity.
- REST API

## Timgiasu model works?

User model:

- User can register in : Normal user or Admin
- User can login in : Normal user or Admin
- CRUD user info

SUbject model:

- Admin can post new model
- Render all subject cards
